ALL COMMANDS

/activites

Gives you a list of activities around Rowan University

/bot

Returns information about the Rowan Chatbot

/campus

Returns information about Rowan campus & surrounding buildings

/course

Provides infomation about rowan courses for the current year

/crisis

Returns information about Rowan crisis resources

/department

Provides infomation about each and all Rowan Departments

/dining

Tells the location of places to eat around rowan

/help

Returns list of available Chatbot commands

/landmarks

Returns map locations of important landmarks around Rowan

/parking

Returns a map location of the parking garages at Rowan

/registration

Shows the date for registration based on the year/season

/university

Returns information about the various colleges and universities at Rowan

/weather

Returns the current weather at Rowan University

/bus

Returns specific details for when the next shuttle bus will be arriving at its stop
